# ENGG*1420 Project - Campus Event Booking System

## How to Compile and Run the Program
1. Clone the repository
```bash
https://github.com/louisnguyenn/ENGG-1420-Final-Project.git
```
2. Open the project in IntelliJ IDEA
3. Locate the main file in 'src/main/java/MainApp.java'
4. Click the 'Run' button in IntelliJ IDE to compile and run the program

## How to execute the Test Suite
1. Locate 'test/java/BookingTests.java'
2. Right-click the file and click "Run 'BookingTests'" to run all 4 of the tests at once
3. To individually run a test, locate a method with the annotation '@Test' above it
4. Click the 'Run Test' button on the left side of the method declaration to run the test
